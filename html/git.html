Git

  • git add: добавление изменений из рабочей области в область подготовки.
  • git rm: удаление файла из репозитория.
  • git mv: перемещение или переименование файла в репозитории.
  • git status: проверка статуса рабочего дерева.
  • git commit: сохранение подготовленных изменений в репозитории.
  • git pull: скачивание изменений и автоматическое их объединение с вашим рабочим деревом.
  • git push: отправка ваших локальных изменений на удалённый сервер.
  • git show: подробный отчёт о конкретном коммите.
  • git branch: управление ветками (создание, удаление, переключение).
  • git clone: получение копии существующего репозитория.
